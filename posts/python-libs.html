Mis librerías favoritas de Python

Python es famoso por su filosofía de "baterías incluidas", pero el verdadero poder del lenguaje reside en su ecosistema de paquetes de terceros (PyPI). A continuación, comparto mis librerías indispensables que utilizo en casi todos los proyectos.

1. Requests: HTTP para Humanos

Hacer peticiones web con la librería estándar urllib es engorroso. Requests simplifica enormemente la interacción con APIs RESTful.

import requests

respuesta = requests.get('https://api.github.com/users/rodrigo47363')
if respuesta.status_code == 200:
    data = respuesta.json()
    print(f"Usuario: {data['login']}")

2. Click: Interfaces de Línea de Comandos

Si estás creando herramientas para la terminal, Click es infinitamente superior a argparse. Te permite crear comandos complejos usando simples decoradores.

import click

@click.command()
@click.option('--count', default=1, help='Número de saludos.')
@click.argument('name')
def hello(count, name):
    for x in range(count):
        click.echo(f"Hola {name}!")

if __name__ == '__main__':
    hello()

3. Pandas: Análisis de Datos

Indispensable si trabajas con CSVs, Excel o bases de datos SQL. Pandas te permite filtrar, agrupar y transformar datos masivos en cuestión de milisegundos.

import pandas as pd

df = pd.read_csv('ventas.csv')
# Filtrar ventas mayores a 100 y agrupar por mes
resumen = df[df['total'] > 100].groupby('mes').sum()
print(resumen)

4. BeautifulSoup4: Web Scraping

Cuando la página web no tiene una API, BeautifulSoup4 (junto con Requests) te permite extraer datos del código HTML de forma estructurada e intuitiva.

Conclusión

Conocer las librerías adecuadas evita reinventar la rueda. Te invito a crear entornos virtuales (python -m venv venv) y experimentar con estas herramientas para llevar tus scripts al siguiente nivel.
